在 PostgreSQL 中，extension 的概念和 Oracle 中的存储过程类似，但 PostgreSQL 的 extension 功能更强大。
- 允许你使用 SQL 创建函数等
- 允许你用 PL/pgSQL 写类似存储过程的代码，创建函数、类型等
- 允许你使用 c 写扩展代码
